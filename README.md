# 🧠 Computer Vision Project - Image Classification

## Description
This project focuses on image classification using deep learning techniques.  
Two models were developed using **PyTorch** and **TensorFlow**, and the best-performing model was selected for deployment.

The application allows users to upload an image and get a predicted class in real time.

---

##  Features
- Image classification using CNN
- Training with PyTorch and TensorFlow
- Model comparison and selection
- Web interface for prediction
- Deployment-ready application

---

##  Models

### 🔹 PyTorch Model
- Built and trained using PyTorch
- Used for experimentation and comparison

### 🔹 TensorFlow Model  (Selected)
- Achieved better accuracy and consistency
- Selected for deployment

---

##  Dataset
- Dataset used: Intel Image Classification Dataset
- Classes:
  - Buildings
  - Forest
  - Glacier
  - Mountain
  - Sea
  - Street
