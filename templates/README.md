# Intel Image Classification Project

## Description
This project is an image classification system using CNN models built with PyTorch and TensorFlow.

## Dataset
Intel Image Classification dataset (6 classes):
- buildings
- forest
- glacier
- mountain
- sea
- street

## Models
- PyTorch CNN model
- TensorFlow CNN model

## Requirements
Install dependencies:
pip install flask tensorflow torch numpy pillow

## How to Run
1. Activate virtual environment
2. Run the app:
python app.py
3. Open browser:
http://127.0.0.1:5000

## Output
Upload an image and the model predicts its class.
